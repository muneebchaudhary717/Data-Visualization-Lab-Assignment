# Data-Visualization-Lab-Assignment
This project analyzes the Titanic dataset using R and R Markdown. Twenty different visualizations were created to explore passenger demographics and survival patterns. The analysis shows that gender and passenger class significantly influenced survival, with females and first-class passengers having higher survival rates.
